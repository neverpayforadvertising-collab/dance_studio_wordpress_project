# 💃 Vacile Dance Studio Website

## 📝 Project
**Professional Dance Studio Website Development with External Booking Integration**

## 🌟 Overview
This project involves designing and developing a modern, energetic, and fully responsive website for Vacile Dance Studio. The website presents the studio’s services, schedule, and contact information while allowing students to book classes through the **Buenonu** booking platform.

The booking system is integrated using embedded elements or booking links so users can register and make payments easily without leaving the website.

## 🎯 Goals
- ✅ Build a clean, modern, and mobile-responsive website  
- ✅ Clearly present studio services and schedule  
- ✅ Integrate an external booking system
- ✅ Provide intuitive navigation and strong call-to-action elements  
- ✅ Apply basic SEO and performance optimization  
- ✅ Enable easy content updates via WordPress  

## ✨ Key Features
- 📱 Responsive design for desktop and mobile  
- 🧭 Clear navigation and user-friendly layout  
- 💳 External booking integration (Buenonu)  
- 🔘 “Book Now” call-to-action buttons  
- ✉️ Contact form and social media links  
- ⚡ Optimized loading speed
